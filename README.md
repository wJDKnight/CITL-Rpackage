# CITL-Rpackage
Here is the R package of CITL. The command line tool is at https://github.com/wJDKnight/CITL/ 
# Requirements
  R packages: bnlearn; snow  
# Installation 
  Use R package: devtools to install CITL.
    
    devtools::install_github("wJDKnight/CITL-Rpackage")
